# Mem0 v2.0 API Compatibility Fix

## Problem
`search_memory` + `get_context` crash. Error: `ValueError: Top-level entity parameters frozenset({'user_id'}) are not supported in get_all(). Use filters={'user_id': '...'} instead.`

## Root Cause
`mem0ai` v2.0.0+ change API. `get_all()` + `search()` need `filters` dict for `user_id`.

## Solution
Update `src/mem0_local_mcp/mem_manager.py`. Move `user_id` to `filters`.

### Before
```python
all_mems = self.memory.get_all(user_id=self.user_id)
results = self.memory.search(query, filters=search_filters)
```

### After
```python
results = self.memory.get_all(filters={"user_id": self.user_id})
filters = {"user_id": self.user_id}
if category:
    filters["category"] = category
results = self.memory.search(query, filters=filters, top_k=limit)
```

## Updated Findings
- **Indentations**: Ensure `return` correctly indented in `with self.lock:`.
- **Inference Stability**: `infer=False` in `memory.add()` store literal. Prevent LLM extraction failures.
